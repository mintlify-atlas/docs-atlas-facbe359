---
title: "Update Customer"
api: "PUT /api/v1/customers/{id}/"
description: "Update a customer's profile information and address"
---

Updates a customer's profile information including personal details and address. Only the customer who owns the account can update their profile.

<Note>
  This endpoint requires authentication and customers can only update their own profiles.
</Note>

## Path Parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the customer to update
</ParamField>

## Request Body

<ParamField body="email" type="string">
  Customer's email address. Must be unique if changed.
</ParamField>

<ParamField body="first_name" type="string">
  Customer's first name
</ParamField>

<ParamField body="last_name" type="string">
  Customer's last name
</ParamField>

<ParamField body="date_of_birth" type="string">
  Customer's date of birth in ISO 8601 format (YYYY-MM-DD)
</ParamField>

<ParamField body="address" type="object">
  Customer's shipping/billing address. Provide this object to create or update the address.
  
  <Expandable title="Address properties">
    <ParamField body="street_address" type="string" required>
      Street address (max 30 characters)
    </ParamField>
    
    <ParamField body="postal_code" type="integer" required>
      Postal code
    </ParamField>
    
    <ParamField body="city" type="string" required>
      City name (max 50 characters)
    </ParamField>
    
    <ParamField body="state" type="string" required>
      State or province (max 50 characters)
    </ParamField>
    
    <ParamField body="country" type="string" required>
      Country name in full (max 30 characters, must be more than 3 characters)
    </ParamField>
  </Expandable>
</ParamField>

```python Python
import requests

url = "https://api.example.com/api/v1/customers/123/"
headers = {
    "Authorization": "Bearer YOUR_TOKEN",
    "Content-Type": "application/json"
}
payload = {
    "email": "customer@example.com",
    "first_name": "John",
    "last_name": "Doe",
    "date_of_birth": "1990-01-15",
    "address": {
        "street_address": "456 Oak Avenue",
        "postal_code": 54321,
        "city": "Los Angeles",
        "state": "California",
        "country": "United States"
    }
}

response = requests.put(url, json=payload, headers=headers)
print(response.json())
```
