## List All Categories

<CodeGroup>
```bash GET
GET /api/v1/categories/
```
</CodeGroup>

### Authentication

This endpoint does not require authentication. It is publicly accessible.

### Query Parameters

<ParamField query="search" type="string">
  Search categories by name
</ParamField>

### Response

<ResponseField name="id" type="integer">
  Unique category identifier
</ResponseField>

<ResponseField name="name" type="string">
  Category name
</ResponseField>

<ResponseField name="slug" type="string">
  URL-friendly category identifier (auto-generated from name)
</ResponseField>

### Request Example

<CodeGroup>
```bash List All Categories
curl -X GET "https://api.example.com/api/v1/categories/" \
  -H "Accept: application/json"
```

```bash Search Categories
curl -X GET "https://api.example.com/api/v1/categories/?search=electronics" \
  -H "Accept: application/json"
```
</CodeGroup>

### Response Example

```json 200 OK
[
  {
    "id": 1,
    "name": "Electronics",
    "slug": "electronics"
  },
  {
    "id": 2,
    "name": "Clothing",
    "slug": "clothing"
  },
  {
    "id": 3,
    "name": "Home & Garden",
    "slug": "home-garden"
  }
]
```

## Get Category Details

Retrieve a specific category along with all its associated products.

<CodeGroup>
```bash GET
GET /api/v1/categories/{slug}/
```
</CodeGroup>

### Authentication

This endpoint does not require authentication. It is publicly accessible.

### Path Parameters

<ParamField path="slug" type="string" required>
  The URL-friendly slug identifier of the category
</ParamField>

### Response

<ResponseField name="id" type="integer">
  Unique category identifier
</ResponseField>

<ResponseField name="name" type="string">
  Category name
</ResponseField>

<ResponseField name="slug" type="string">
  URL-friendly category identifier
</ResponseField>

<ResponseField name="product_set" type="array">
  Array of all products in this category
  
  <Expandable title="Product Object">
    <ResponseField name="id" type="integer">
      Unique product identifier
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Product name
    </ResponseField>
    
    <ResponseField name="category" type="string">
      Category name
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Product description
    </ResponseField>
    
    <ResponseField name="base_price" type="string">
      Product base price or lowest variant price
    </ResponseField>
    
    <ResponseField name="rating" type="float">
      Average customer rating
    </ResponseField>
    
    <ResponseField name="total_stock_level" type="integer">
      Total available stock
    </ResponseField>
    
    <ResponseField name="is_available" type="boolean">
      Availability status
    </ResponseField>
    
    <ResponseField name="media" type="array">
      Product media (images/videos)
    </ResponseField>
    
    <ResponseField name="product_offers" type="array">
      Active offers for the product
    </ResponseField>
  </Expandable>
</ResponseField>

### Request Example

<CodeGroup>
```bash cURL
curl -X GET "https://api.example.com/api/v1/categories/electronics/" \
  -H "Accept: application/json"
```

```python Python
import requests

url = "https://api.example.com/api/v1/categories/electronics/"
response = requests.get(url)
category = response.json()

print(f"Category: {category['name']}")
print(f"Number of products: {len(category['product_set'])}")
```

```javascript JavaScript
fetch('https://api.example.com/api/v1/categories/electronics/')
  .then(response => response.json())
  .then(category => {
    console.log(`Category: ${category.name}`);
    console.log(`Products: ${category.product_set.length}`);
  });
```
</CodeGroup>

## List Products by Category

Get a paginated list of products filtered by category slug.

<CodeGroup>
```bash GET
GET /api/v1/products/categories/{slug}/
```
</CodeGroup>

### Authentication

This endpoint does not require authentication. It is publicly accessible.

### Path Parameters

<ParamField path="slug" type="string" required>
  The URL-friendly slug identifier of the category
</ParamField>

### Query Parameters

<ParamField query="search" type="string">
  Search products within this category
</ParamField>

<ParamField query="is_available" type="boolean">
  Filter by availability status
</ParamField>

<ParamField query="store" type="integer">
  Filter by store ID
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Number of products per page
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Pagination offset
</ParamField>

### Request Example

<CodeGroup>
```bash Basic Request
curl -X GET "https://api.example.com/api/v1/products/categories/electronics/" \
  -H "Accept: application/json"
```

```bash With Filters
curl -X GET "https://api.example.com/api/v1/products/categories/electronics/?is_available=true&limit=20" \
  -H "Accept: application/json"
```
</CodeGroup>

### Response Example

```json 200 OK
{
  "count": 45,
  "next": "https://api.example.com/api/v1/products/categories/electronics/?limit=10&offset=10",
  "previous": null,
  "results": [
    {
      "id": 1,
      "name": "Premium Wireless Headphones",
      "category": "Electronics",
      "base_price": "149.99",
      "rating": 4.5,
      "is_available": true,
      "media": [...],
      "product_offers": [...]
    }
  ]
}
```

## Notes

- Categories are ordered alphabetically by name
- Category slugs are automatically generated from the category name when created
- Slugs are URL-safe and lowercase (e.g., "Home & Garden" becomes "home-garden")
- Products within categories are ordered by `total_sold` (descending) and then by `rating` (descending)
- The category lookup by slug is case-sensitive
